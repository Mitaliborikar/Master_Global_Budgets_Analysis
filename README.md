# 🌍 Global Master Budget Analysis

## 📖 Overview

This project analyzes historical government budget data from **45 countries over 80 years** using **Python** and **Looker Studio**. The goal is to understand how governments allocate budgets across different sectors, identify long-term spending trends, and build an interactive dashboard for financial analysis and decision-making.

---

## 🎯 Project Objectives

* Analyze government budget allocation across countries.
* Compare sector-wise government spending.
* Identify long-term expenditure trends.
* Evaluate spending priorities in different sectors.
* Develop an interactive dashboard for data-driven insights.

---

## 💼 Business Problem

Governments allocate budgets across multiple sectors such as education, healthcare, infrastructure, defence, agriculture, and social welfare. Understanding how these allocations vary across countries and over time helps policymakers, researchers, and analysts identify spending priorities, compare financial strategies, and support informed decision-making.
